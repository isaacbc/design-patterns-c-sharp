# Design Patterns in C#

A personal **study** repository. It gathers **lecture notes** and hands-on material from courses in **Software Engineering** and **Program Construction Techniques**, with emphasis on **design patterns**, object-oriented programming, and related practices.

The content is informed by coursework and by readings such as:

- *Mastering Design Patterns in C#: A Beginner-Friendly Guide, Including OOP and SOLID Principles*
- *Implementation Patterns* — Kent Beck
- *Object-Oriented Software Construction* — Bertrand Meyer

Examples are written in **C#** and are organized so that **pieces work together** in a single runnable project, to support **integrated understanding** of the concepts—not isolated snippets only.

## Purpose

- Consolidate what I learn in class and from books.
- Experiment with patterns and OOP/SOLID ideas in code.
- Keep a clear trail of notes and runnable examples for review.

## Repository layout

- **`Studying-Design-Patterns/`** — main console application (`design-patterns-c-sharp.sln`).
- **`Studying-Design-Patterns/OOP/`** — object-oriented topics (e.g. encapsulation, abstraction, inheritance, polymorphism, coupling) with paired markdown notes and `.cs` examples where present.

## Requirements

- [.NET SDK](https://dotnet.microsoft.com/download) compatible with **.NET 10** (see `Studying-Design-Patterns/Studying-Design-Patterns.csproj`).

## Build and run

From the repository root:

```bash
dotnet build
dotnet run --project Studying-Design-Patterns
```
